Scratch

Scratch — визуально-блочная среда программирования, созданная для детей и подростков.
Цель программирования на Scrach познакомить ребёнка с главными фундаментальными законами программирования, развить логическое мышление и научить думать как программисты.

Программирование на Scratch я сравниваю с изучением правил дорожного движения. Зная правила можно безопасно и комфортно ездить на любом виде транспорта, которым умеешь управлять. Также и основы программирования, зная их, ребёнок сможет создавать логичные по структуре программы, неважно на каком языке он будет их писать.

Front-End

На этом курсе ребёнок изучит синтаксис и правила языков – HTML, CSS и JavaScript. Что это такое? Это языки с помощью которых создаются сайты. Ниже официальные определения для каждого языка.

HTML — стандартизированный язык разметки документов для просмотра веб-страниц в браузере.
CSS — формальный язык описания внешнего вида документа, написанного с использованием языка разметки.
JavaScript — мультипарадигменный язык программирования. Поддерживает объектно-ориентированный, императивный и функциональный стили.

Изучение будет проходить на практике. Каждое занятие ученик будет создавать небольшой сайт или часть сайта. Также ребёнок научиться публиковать сайты в интернет на бесплатный хостинг.

Roblox Studio

Roblox — многопользовательская онлайн-платформа, которая позволяет пользователям играть в созданные другими пользователями игры и создавать свои собственные.

Эта игра очень популярна среди детей до 12 лет. Однако немногие знают, что на этой платформе можно создавать свои собственные игры, публиковать их и даже зарабатывать деньги!
На этом курсе ребёнок знакомиться с функционалом платформы, 3D моделированием и затем программированием на языке Lua.

Lua — это процедурный динамически типизированный модульный язык с автоматическим управлением памятью.
